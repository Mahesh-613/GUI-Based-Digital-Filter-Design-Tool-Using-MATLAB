###### 🚀 Built a GUI-Based Digital Filter Design Tool Using MATLAB

###### 

I recently completed a project to develop an interactive, all-in-one GUI for the design, analysis, and validation of Infinite Impulse Response (IIR) filters.



When we hear the word "filter," many people outside the electronics domain may associate it with photo editing or image enhancement. However, in electronics, a filter is a circuit or system that selectively passes or attenuates specific frequency ranges within a signal.



Digital filters are fundamental building blocks of modern Digital Signal Processing (DSP) systems, powering applications ranging from audio processing and telecommunications to biomedical instrumentation. However, designing and comparing different filter types often requires switching between multiple tools and manual calculations.



To address this, I developed a MATLAB-based GUI that brings the entire workflow into a single platform.



🔹 Key Features



🎛️ Unified Filter Design Interface – Configure Low-Pass, High-Pass, Band-Pass, and Band-Stop filters through a user-friendly interface.



📊 Comparative Analysis – Simultaneously visualize the magnitude responses of Butterworth, Chebyshev Type-I, Chebyshev Type-II, and Elliptic filters to quickly evaluate performance trade-offs.



✅ Stability Verification – Automatically generate Z-plane pole-zero plots and identify unstable filter configurations.



💻 Hardware Resource Estimation – Calculate the exact number of adders, multipliers, and delay elements required for Direct Form-I and Direct Form-II implementations.



🎵 Filter Validation – Apply the designed filters to audio signals and evaluate their real-world performance.



Through this project, I gained deeper insights into IIR filter design, frequency response analysis, stability considerations, and hardware implementation aspects of digital signal processing.
